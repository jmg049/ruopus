\section{Python API Reference}\label{sec:python_guide}

\subsection{Introduction}\label{subsec:python_intro}

Python bindings for \texttt{opus\_native} are planned but not yet released. This
section is reserved for the complete Python reference: installation, the
encoder and decoder objects, streaming and file (Ogg Opus) helpers, and worked
examples that apply the concepts from the preceding sections in practice.

\begin{importantnote}
  This chapter is a placeholder. The bindings (a \texttt{PyO3} extension module
    mirroring the Rust \code{OpusEncoder}/\code{OpusDecoder} surface, with NumPy
  buffer-protocol interop for PCM) are under construction. Until they ship, use
  the Rust API documented in \cref{sec:opus_layer} directly, or call the C
  reference \texttt{libopus} through an existing binding. The remainder of this
  section is intentionally left blank and will be filled in alongside the
  first binding release.
\end{importantnote}
